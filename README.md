# Band Name Generator

A simple beginner-friendly Python project that generates a fun band name based on user input.

This program asks:
- What city you grew up in
- What your pet's name is

It then combines both answers to suggest a band name.

## Project Goal

This project is part of my Python learning journey and helps practice:
- Taking user input with `input()`
- Storing values in variables
- Displaying output with `print()`
- Using f-strings for readable string formatting

## How It Works

When you run the script, the program:
1. Welcomes the user
2. Asks for the name of the city they grew up in
3. Asks for their pet's name
4. Prints a possible band name using both answers

## Example

```text
Welcome to the Band Name Generator!
What's the name of the city you grew up in?
Miami
What's your pet's name?
Rocky
Your band name could be Miami Rocky.
```

## Run the Project

Make sure Python is installed, then run:

```bash
python band_name_generator.py
```
